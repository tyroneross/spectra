# Solo production + measurement (quick card)

Full detail: `SOURCE-GUIDE.md` Parts VI–VII + Appendix. For a solo builder, production
efficiency is the constraint, not budget (a pro workflow is <$300 in tools).

## Film once, publish many (1 session → a whole set)
Every anchor recording yields: 1 long-form (YouTube/demo, 5–15m) · 2–3 MOFU clips (2–5m) ·
5–10 TOFU social clips (15–90s) · 1 thumbnail frame · 1 GIF/loop (README/X). Spectra's
polish pipeline re-renders one source into each aspect/duration/preset — see spectra-production-map.md.

## Lean stack essentials
iPhone 4K + gimbal (~$100) · lavalier/USB mic ($50–100, **critical**) · CleanShot X/OBS
screen capture · Descript/Adobe Podcast (audio cleanup, captions) · Opus Clip/Captions.ai
(repurpose to shorts) · CapCut (edit) · Canva/ScreenKit (thumbnails/frames).

## Recording basics
Smallest room + close mic + AI noise removal (this alone makes you competitive) · face a
window (diffuse with a sheet) · big fonts + high-contrast theme + notifications off + clean
demo data · camera at eye level.

## Measurement — watch depth = intent
- 25% → mild curiosity → serve more awareness content
- 50–75% → moderate interest → serve demo/feature
- 75–95% → high intent → serve testimonials/case studies/direct CTA
- 95–100% → conversion-ready → direct offer/trial

| Stage | Metrics |
|---|---|
| TOFU | views, watch time, subs, CTR to site |
| MOFU | demo requests, trial signups, email CTR, time on page |
| BOFU | win rate (video shared vs not), deal cycle, testimonial view-to-close |

## Cadence
Monthly: review top/bottom by watch time + CTA CTR. Quarterly: refresh BOFU (new metrics,
quotes, current UI). Every release: ship a changelog/release video (highest-ROI, low-effort).

## Checklists (Appendix in SOURCE-GUIDE.md)
- **Pre**: goal · audience · type · one-sentence message · story spine · script (tech-reviewed if dev) · clean demo env.
- **Prod**: clean audio · lighting · constant 30fps · eye-level · first-5s planned (no logo/intro).
- **Post**: burned-in captions · on-screen key messages · info every 2–3s · one CTA · exported per aspect (16:9/1:1/9:16).
- **App Store**: H.264 High (not HEVC) · constant 30fps · correct res · 15–30s · AAC stereo 48kHz (even if silent) · yuv420p · .mp4 faststart · no frames/pricing/competitor refs.
