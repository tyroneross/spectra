# Product-type playbooks (quick card)

Full detail: `SOURCE-GUIDE.md` Part IV. Production recipes: `spectra-production-map.md`.

## AI agents (product is a behavior/output, not a UI)
- Structure: hook (specific problem) → name+position → **dream outcome (show the output)** →
  2–3 use cases in existing workflows → CTA.
- Show trigger → action → output loop in real time; real outputs (not mocked);
  narrated walkthrough + callouts for multi-step. **Avoid abstract AI imagery** (glowing
  neural nets, floating particles) — devs/buyers ignore it.
- Distribute: LinkedIn (founder POV of outputs), YouTube (workflow/integration), PH launch,
  README demo GIF, X (impressive-output screen recordings).

## Browser plugins / extensions
- Record in a real browser with the plugin active; before/after; highlight the value moment
  (popup/highlight/shortcut/transform); tight 45–60s. Chrome Web Store = App Store logic (real UI, 15–30s).
- Distribute: PH, Chrome Web Store, LinkedIn, X, relevant subreddits, Show HN.

## iOS / iPhone (strongest emotional hook, fastest value)
- App Store preview per specs; structure: visual hook (3s) → core use case → optional wow → icon/name.
- Organic (Reels/TikTok/Shorts): app in a relatable real scenario; problem-agitate-solve;
  founder-led > polished ad; consistency (12–24mo) is the path to $20K+ MRR.
- Paid: test 3–5 hooks against the same demo; spend on the hook, not production.

## Native macOS (between dev tool and consumer)
- App Store (Mac) landscape 1920×1080; show keyboard shortcuts, native patterns (menu bar,
  context menus), macOS integration (Spotlight, Notification Center, Quick Look).
- **Show Mac-native feel** — strong signal to power users.
- Distribute: r/MacApps, MacRumors, MPU, PH, X (Mac dev community), YouTube.

## Apple Watch (most constrained — value in the smallest surface)
- Show the glance moment; core interaction (complication → tap → done, 1–3 taps);
  "no phone needed" / quick-access value. App Store preview 15–30s portrait, real UI, no frames.
- Marketing (outside App Store): show on a wrist in real life (workout/commute/meeting).

## Web app (B2B)
- SMB: explainer (60–90s hero) + demo (3–5m). Enterprise: use-case + case study (3–8m),
  ROI, implementation walkthrough. Homepage/LinkedIn/email + sales deck.
