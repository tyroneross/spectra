# Color (quick card)

Full prose + risk notes: `SOURCE-GUIDE.md` Part II.

## 60-30-10 rule (per frame)
- **60% primary (neutral):** background — white, light gray, charcoal, or matte black.
- **30% secondary (brand):** your core brand color — recognition + cohesion.
- **10% accent (CTA/highlight):** high-contrast pop color on the one thing the viewer must look at.

## Core color → emotion map
| Color | Emotion | Use for | Risk |
|---|---|---|---|
| Deep/Royal Blue | Trust, stability | Enterprise B2B, security, infra | Brand-blends — every SaaS uses it |
| Electric Blue/Indigo | Intelligence, future | AI agents, ML, dev platforms | Cold/sterile without warm accents |
| Emerald/Neon Green | Growth, success | CTAs, completion states, fintech | Never for error/warning states |
| Near-Black/Charcoal | Authority, premium | Dev tools, CLI, macOS native, AI | Check contrast on long dark-mode text |
| White/Off-White | Clarity, simplicity | Minimalist consumer, clean demos | Flat without strong type/accent |
| Purple/Violet | Creativity, premium | Design tools, premium AI tier | Reads exclusive — differentiator, not core |
| Orange | Energy, approachable | Indie/prosumer, CTA buttons | Playful — avoid pure enterprise security |
| Red | Urgency, danger | CTAs only, errors | Never as primary B2B product color |

## Dark vs. light mode
- **Dark:** default for dev tools, AI platforms, CLI, native macOS/Watch — signals premium tech, makes data pop.
- **Light:** better for long-form text, consumer iOS, non-technical B2B demos.
- Match how the audience actually uses the product — never pick light because it photographs better.

## Color by funnel moment
- **Awareness:** vibrant, saturated, high-energy — interrupts scrolling.
- **Evaluation/demo:** cleaner, balanced — focus goes to the product UI, not graphics.
- **Conversion (CTA frame):** all distraction removed; only the CTA is high-contrast.
